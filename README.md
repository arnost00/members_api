# URL Structure

The URL structure for accessing the API follows the pattern:

```
/api/<version>/<endpoint>
```

**Components:**

- `/api/` - Base path for all API requests.
- `<version>` - API version number (e.g., `1`).
- `<endpoint>` - Specific API endpoint (e.g., `/clubs`, `/spe/races`).
